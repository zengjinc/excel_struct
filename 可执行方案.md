# Excel结构生成工具 - 可执行方案

## 项目概述

本项目实现了一个通过JSON配置文件生成Excel结构化表格的工具，支持Erlang和Lua函数配置，用于游戏配置表管理。

## 核心功能

1. **配置驱动生成**: 通过JSON配置文件定义Excel结构
2. **多Sheet支持**: 一个配置文件可生成包含多个Sheet的Excel
3. **Erlang/Lua函数配置**: 支持完整的ERL_FUN和LUA_FUN配置
4. **数据与结构分离**: 修改配置文件只修改结构，保留VALUE数据
5. **灵活的输出**: 支持自定义输出路径和目录

## 技术架构

### 模块设计

```
generate_excel.py (主入口)
    ↓
config_parser.py (配置解析)
    ↓
excel_generator.py (Excel生成)
    ↓
target/*.xlsx (输出文件)
```

### 核心模块

#### 1. config_parser.py - 配置解析模块

**职责**: 解析JSON配置文件，验证配置有效性

**核心类**:
- `ConfigParser`: 主解析器类
- `ExcelConfig`: Excel配置数据类
- `SheetConfig`: Sheet配置数据类
- `ErlFun`: Erlang函数配置类
- `LuaFun`: Lua函数配置类
- `Field`: 字段配置类

**主要方法**:
- `parse_file(file_path)`: 从文件解析配置
- `parse_dict(data)`: 从字典解析配置
- `validate(config)`: 验证配置有效性

#### 2. excel_generator.py - Excel生成模块

**职责**: 根据配置生成Excel文件

**核心类**:
- `ExcelGenerator`: Excel生成器类

**主要方法**:
- `generate(config, output_path)`: 生成Excel文件
- `_generate_sheet(sheet)`: 生成单个Sheet
- `_write_erl_name()`: 写入ERL_NAME行
- `_write_erl_include()`: 写入ERL_INCLUDE行
- `_write_erl_funs()`: 写入ERL_FUN行
- `_write_lua_name()`: 写入LUA_NAME行
- `_write_lua_funs()`: 写入LUA_FUN行
- `_write_pre_format()`: 写入PRE_FORMAT行
- `_write_fields()`: 写入FIELD行
- `_write_notes()`: 写入NOTE行
- `_write_values()`: 写入VALUE行
- `_apply_styles()`: 应用样式

#### 3. generate_excel.py - 主脚本入口

**职责**: 命令行接口，协调各模块工作

**功能**:
- 解析命令行参数
- 读取配置文件
- 验证配置
- 调用生成器
- 输出结果

## 配置文件格式

### 基本结构

```json
{
  "output_file": "输出文件名.xlsx",
  "sheets": [
    {
      "name": "Sheet名称",
      "erl_name": "data_module.erl",
      "erl_include": ["common.hrl"],
      "erl_funs": [...],
      "lua_name": "config_module.lua",
      "lua_funs": [...],
      "fields": [...],
      "values": [...]
    }
  ]
}
```

### 配置项详解

#### Sheet级别配置

| 配置项 | 类型 | 必需 | 说明 |
|--------|------|------|------|
| name | string | 是 | 工作表名称 |
| erl_name | string | 是 | Erlang文件名 |
| erl_include | array | 否 | 需要包含的头文件 |
| erl_funs | array | 否 | Erlang函数配置 |
| lua_name | string | 是 | Lua文件名 |
| lua_funs | array | 否 | Lua函数配置 |
| pre_format | object | 否 | 预格式化配置 |
| fields | array | 是 | 字段定义 |
| values | array | 是 | 数据值 |

#### ERL_FUN配置

| 字段 | 类型 | 必需 | 说明 |
|------|------|------|------|
| name | string | 是 | 函数名（小写+下划线） |
| params | object | 是 | 参数配置 |
| return | string | 否 | 返回类型 |
| when | string | 否 | when条件 |
| note | string | 否 | 函数注释 |
| fun_note | array | 否 | 字段注释 |
| filter | object | 否 | 过滤配置 |

#### LUA_FUN配置

| 字段 | 类型 | 必需 | 说明 |
|------|------|------|------|
| name | string | 是 | 函数名（双驼峰） |
| params | object | 是 | 参数配置 |
| default | object | 否 | 默认值 |
| return | string | 否 | 返回类型 |
| sub_key | array | 否 | 子键 |
| key_split | string | 否 | 键分隔符 |
| filter | object | 否 | 过滤配置 |

#### FIELD配置

| 字段 | 类型 | 必需 | 说明 |
|------|------|------|------|
| name | string | 是 | 字段名 |
| note | string | 是 | 字段说明 |

## 使用流程

### 1. 创建配置文件

在 `struct/` 目录下创建JSON配置文件

### 2. 运行生成命令

```bash
python generate_excel.py struct/your_config.json
```

### 3. 查看生成的Excel文件

生成的文件位于 `target/` 目录

## 命令行参数

```
usage: generate_excel.py [-h] [-o OUTPUT] [-d OUTPUT_DIR] config

positional arguments:
  config                配置文件路径 (JSON格式)

optional arguments:
  -h, --help            显示帮助信息
  -o OUTPUT, --output OUTPUT
                        输出文件路径 (可选，默认使用配置文件中的output_file)
  -d OUTPUT_DIR, --output-dir OUTPUT_DIR
                        输出目录 (可选，默认为target目录)
```

## 示例

### 示例1: 简单配置

## 故障排查

### 常见问题

1. **配置文件格式错误**
   - 检查JSON语法
   - 确保使用UTF-8编码

2. **字段不匹配**
   - 检查values数组长度与fields数量是否一致
   - 确保数据类型正确

3. **文件路径问题**
   - 使用绝对路径或相对路径
   - 确保目录存在

4. **编码问题**
   - 确保配置文件使用UTF-8编码
   - 注意特殊字符处理
